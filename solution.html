１ - 密度の計算

密度とは単位体積あたりの質量のこと。
g/㎤という単位からもわかるように、密度を求める式は

密度[g/㎤] = 物質の質量[g] / 物質の体積[㎤]

になります。
なので、例えば～新２年生 春の陣～の時の数字で計算すると、

質量51g、体積30㎤の物体の密度はいくらか。
▼
51[g] / 30[㎤] = 1.7[g/㎤]

つまり、答えは1.7g/㎤となります。密度は単位で式を覚えるのが最も良いと思います。

１年 - 質量パーセント濃度の計算

濃度という存在を考えた人が悪いと思います。重罪です。もっといい方法はなかったのでしょうか。
大変分かりにくいですが、質量パーセント濃度を求める式は

質量パーセント濃度 = (溶質の質量[g] / 溶液の質量[g]) × 100

になります。溶液 = 溶媒 + 溶質なので、
質量パーセント濃度は液体に物質をぶちこんだものの中で、
そのぶち込んだものがどれだけの割合を占めているかを言います。
100を掛けることもお忘れなく。

水100gに食塩25gをとかした食塩水の質量パーセント濃度はいくらか。
▼
(25[g] / (100[g] + 25[g])) × 100
= (25[g] / 125[g]) × 100
= 0.2 × 100 = 20[%]

つまり、答えは20%になるわけです。すごくわかりづらい。厳しいって。

１年 - 振動数の計算

音とは空気を伝わる波のことであり、振動数は一秒あたりに振動した回数のことです。
単位であるHzは日常生活でもよく聞きます。

振動数[Hz] = 振動した回数[回] / 振動するのにかかった時間[秒]

で求めることができます。向きの流れを変える電流、交流の周波数などにも応用されます。

1回の振動が1. 6秒かかる波の振動数。
▼
1[回] / 1.6[秒] = 0.625[Hz]

なので、答えは0.625Hzになります。計算がやりやすくていいですね。最高かよ。

２年 - オームの法則の計算

オームの法則とは、電流・電圧・抵抗の３つの関係を表した式です。

抵抗 = 電圧 / 電流
電流 = 電圧 / 抵抗
電圧 = 電流 × 抵抗

が成り立ちます。（私はV[電圧]=R[抵抗]I[電流] ● VRIの鰤で覚えてます。）

5Ωの抵抗に6Vの電圧をかけるとき、流れる電流は何Aか。
▼
6[V] / 5[Ω] = 1.2[A]

よって、答えは1.2Aとなります。オームの法則くらいはパッと出てくるようにしたいです。

２年 - 電力から電流を求める計算

電力とは......言わずもがな電力です。
単位であるWは電子レンジなど、様々なところで見かけます。

電力[W] = 電流[A] × 電圧[V]

最も簡単だと思います。完全にサービス問題です。こういうのでいいんだよ、こういうので。

50Wの電球を100Vの電源につないで使用するとき、流れる電流は何Aか。
▼
50[W] / 100[V] = 0.5[A]

よって、答えは0.5Aとなります。ありがたいですね。

２年 - 電力量の計算

電力と電力量の違いは、時間が関わるか、関わらないかです。
Jという単位は本当にいろんなところに登場してきます。煩わしい。

電力量[J] = 電力[W] × 時間[秒]

計算すると、とにかく数が巨大になります。これ考えた人は小学生なのかな。

70Wの電球を2時間使ったときの電力量は何Jか。
▼
70[W] × (2 × 3600)[秒] = 504000[J]

よって、答えは504000Jになります。位を間違えたりがないように。

２年 - 圧力の計算

おそらくは「多分これ地学でやる内容じゃなくて草」第一位です。
hPaという単位に罪はありませんが、１気圧 = 1013hPaという事実が気持ち悪すぎます。

圧力[Pa] = 物体が与える力[N] / 接する面積[㎡]

㎠を㎡に直さなければいけないところがやはり難点です。

3600gで、底面積が5㎠の物質を床に置いたとき、床にかかる圧力はいくらか。ただし、100gの物質にかかる重力の大きさを1Nとする。
▼
36[N] / 0.0005 = 72000[Pa]

よって、答えは72000Paとなります。位を本当に間違えやすいので、気を付けてください。

２年 - 湿度の計算

空気中に含むことのできる最大の水蒸気の量を飽和水蒸気量といいます。
「湿度知りてー！」と常日頃から考える人外がこの世にいるとは考えられません。計算が大変すぎます。

湿度[%] = (空気中の水蒸気量[g] / 飽和水蒸気量[g]) × 100

「コップの表面が曇り～」だとかの問題でよく使うのでやはりしっかり覚えなければ。m³で合わせることもお忘れなく。

気温15℃で1m³中に8gの水蒸気をふくんでいる空気の湿度は何％か。ただし、15℃のときの1m³中に含むことのできる最大の水蒸気量は12.8g/m³とする。
▼
(8[g] / 12.8[g]) × 100
= 0.625 × 100 = 62.5[%]

よって、答えは62.5%になります。　ああ　つかれたなあ。

２年 - 化学反応式

化学反応式とはそのままの通り、化学反応の様子を式にあらわしたものです。
酸化銀の熱分解が分かりやすい例だと思いますが、

酸化銀 → 銀 + 酸素
Ag₂O[酸化銀] → Ag[銀] + O₂[酸素]
2Ag₂O[酸化銀] → 4Ag[銀] + O₂[酸素]

のように、左右の原子の数を合わせる必要があります。
上の場合、最終的に 銀が４つ 酸素が２つ で左右が等しくなっています。
また、式とはいってもイコールではなく矢印であらわす必要があります。

物質の化学式は残念ながら、ひとつひとつ覚える必要があります。
